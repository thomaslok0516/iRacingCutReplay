# iRacing 视频剪辑工具

自动检测并移除视频中包含"REPLAY"字样的片段，保留纯净的游戏画面。

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用步骤

1. **选择输入视频**：点击"选择文件"按钮，选择要处理的视频文件

2. **选择输出文件夹**：点击"选择文件夹"按钮，选择保存剪辑后视频的文件夹

3. **设置文件前缀**（可选）：输入文件前缀，例如 `race`，生成的文件将是 `race_001.mp4`, `race_002.mp4` 等

4. **配置ROI区域**（可选，推荐）：
   - 默认ROI：X=600, Y=20, 宽度=200, 高度=20
   - 可以点击"预览视频帧"查看并调整ROI区域
   - 点击"应用ROI"保存设置

5. **选择检测模式**：
   - **快速模式**：只使用方法3+PSM6,7，速度快
   - **全面模式**：使用3个预处理方法×PSM6,7,11，准确率高

6. **开始处理**：
   - 点击"分析视频"先分析，再点击"开始剪辑"
   - 或直接点击"一键分析并剪辑"自动完成所有步骤

## 输出结果

每个保留的片段会生成一个独立的MP4文件，文件名格式：`{前缀}_{序号}.mp4`

例如：`race_001.mp4`, `race_002.mp4`, `race_003.mp4`

## 注意事项

- 需要安装 Tesseract OCR
- 需要安装 ffmpeg（用于保留音频）
- 建议设置ROI区域以提高识别准确率和速度
